Zadatak 5.

Zadani - NKA pretvoriti u minimalni DKA.

- okruženje ( q0 ) = { q0, q1, q2, q3 }
- okruženje ( q1 ) = { q1, q2, q2 }
- okruženje ( q2 ) = { q2 }
- okruženje ( q3 ) = { q3 }

dobije se NKA

pretvorba NKA u DKA
- uvode se složena stanja oblika { qi0, qi1, ... , qim }
- prijelazi se ostvaruju unijom prijelaza podstanja

minimizacija DKA
